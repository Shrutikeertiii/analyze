Project Overview

This project demonstrates a robust data processing pipeline for converting raw Excel data into a structured JSON output, utilizing Python with Pandas. The entire workflow is automated through GitHub Actions, ensuring code quality and continuous deployment of results.

Key Technologies

  • Python 3.11+
  • Pandas 2.3+
  • Ruff Linter
  • GitHub Actions
  • Tailwind CSS (for this dashboard)

Data Flow

  1. data.xlsx (source) is converted to data.csv.
  2. execute.py processes data.csv.
  3. Generates result.json as output.

Generated Results

The result.json file, generated by the execute.py script, contains the processed data. This file is automatically published to GitHub Pages with every successful build.

View latest result.json (Live)

Automation Workflow

Our CI/CD pipeline, defined in .github/workflows/ci.yml, automates testing, processing, and deployment:

  • Lints Python code with Ruff.
  • Executes execute.py to generate result.json.
  • Publishes result.json to GitHub Pages for easy access.

This ensures consistent quality and up-to-date data processing outcomes.
